# 基金实时估值查询系统

一个基于Vue 3的纯前端基金实时估值查询单页应用，完美适配电脑端和移动端。

## 功能特性

- 📊 **实时数据**：通过JSONP接口获取基金实时估值数据
- 📱 **响应式设计**：完美适配电脑端和移动端
- 🔄 **自动刷新**：支持手动刷新和自动定时刷新（60秒）
- 💾 **本地存储**：自动保存用户添加的基金列表
- 🎯 **批量操作**：支持批量选择和删除基金
- ⚡ **高性能**：基于Vue 3的响应式系统，数据更新高效

## 技术栈

- **前端框架**：Vue 3.4.0
- **构建工具**：Vite 5.0.0
- **HTTP请求**：原生JSONP实现
- **样式**：纯CSS（Flexbox + Grid + 媒体查询）
- **数据持久化**：localStorage

## 使用说明

### 添加基金

1. 点击"增加基金"按钮
2. 输入6位基金代码（如：013275）
3. 点击"确定"或按回车键

### 删除基金

1. 勾选要删除的基金（支持全选）
2. 点击"删除选中"按钮

### 刷新数据

- **手动刷新**：点击"刷新最新数据"按钮
- **自动刷新**：开启"自动刷新"开关，系统每60秒自动刷新一次

## 数据接口

系统使用以下JSONP接口获取基金数据：

```
https://fundgz.1234567.com.cn/js/{基金代码}.js?rt={时间戳}
```

返回数据格式：
```javascript
jsonpgz({
  "fundcode": "013275",
  "name": "富国中证煤炭指数(LOF)C",
  "jzrq": "2026-02-03",
  "dwjz": "1.9450",
  "gsz": "2.0853",
  "gszzl": "7.21",
  "gztime": "2026-02-04 11:30"
});
```

## 注意事项

1. **编码问题**：接口返回的基金名称可能存在编码问题，系统已做UTF-8处理
2. **网络要求**：需要能够访问基金数据接口的网络环境
3. **数据延迟**：基金数据存在一定的延迟，实时性仅供参考
4. **移动端适配**：表格在移动端支持横向滚动，确保数据完整显示

## 开发说明

### 核心实现

- **JSONP处理**：使用动态script标签实现JSONP请求，支持超时和错误处理
- **数据管理**：基于Vue 3的响应式系统管理基金列表状态
- **定时器管理**：自动刷新功能支持开关，组件卸载时自动清理定时器
- **本地存储**：使用localStorage持久化用户配置

### 样式设计

- **响应式布局**：使用Flexbox和Grid实现自适应布局
- **移动端优化**：通过媒体查询优化小屏幕显示效果
- **现代设计**：采用渐变背景、圆角、阴影等现代UI元素
